Frontend

This demonstrates that Fetch does not need to set the credentials property in order to send Authorization headers.

Please run both servers at the same time. The server will log a the Authorization header when you press each of these buttons.

Note that credentials are set to omit deliberately to demonstrate this fact.

Also note that Access-Control-Allow-Credentials is not required on the external server for Fetch to Authorization headers across. It needs Access-Control-Allow-Headers instead.
